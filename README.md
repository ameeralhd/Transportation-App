# Transportation App

A full-stack transportation booking application built with React, Node.js, and SQLite.

## Features
- **User Booking**: Search and book bus tickets and rental cars.
- **Admin Dashboard**: Manage schedules, inventory, and approvals.
- **Check-in**: Online check-in system for passengers.
- **Inspiration**: curated travel destinations.

## Tech Stack
- **Frontend**: React, Vite
- **Backend**: Node.js, Express
- **Database**: SQLite

## Getting Started

### Installation
1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    cd client && npm install
    cd ..
    ```

### Running the App
```bash
npm start
```
This will start both the backend server and the frontend client.
